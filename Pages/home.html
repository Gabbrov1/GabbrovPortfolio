About This Site

This is my third portfolio, built after two previous attempts that were scrapped for being over-engineered. The first was written in Blazor and hosted on Azure. It worked, but the hosting costs for such a simple site were hard to justify. The second used Astro, React, and MSSQL, a stack that introduced far more complexity than the project ever needed.

The solution was to go back to basics. This version is written in vanilla HTML, CSS, and JavaScript with no frameworks, no build pipeline, and no dependencies. The codebase is lean and easy to maintain, and the site is statically hosted on GitHub Pages at zero cost.

The frontend is simple, but the site does integrate with a small backend. Images are stored in an AWS S3 bucket and served through a Lambda function exposed via API Gateway. This keeps assets decoupled from the static site and avoids exposing the bucket publicly. Projects are fetched at runtime from the GitHub API, so the portfolio stays current without any manual updates.

The result is a site that is fast, cheap to run, and easy to extend. Built with the same principles I try to apply to my work: simplicity, clarity, and purpose.
